MediaPipe Computer Vision Projects

This repository contains multiple real-time computer vision projects built using Python, OpenCV, and MediaPipe.

The projects demonstrate different AI capabilities such as face detection, hand tracking, body pose detection, motion tracking, and virtual drawing.

These projects are useful for learning Computer Vision, AI interaction, and real-time camera processing.

🧠 Projects Description
✋ Air Canvas (air_canvas.py)

Allows drawing on screen using hand gestures detected via MediaPipe.

😀 Face Detection (face1.py, face2.py)

Detects faces in real time using a webcam.

🧍 Pose Detection (pose_det.py)

Tracks human body keypoints for posture analysis.

👋 Hand Landmark Detection (hand_landmark_detection.py)

Detects hand landmarks and finger movements.

🧠 Face + Hand Tracking (face_hand_landmark.py)

Detects both face and hands simultaneously.

🚶 Real Time Body Detection (real_time_body_detection.py)

Tracks body movement using webcam input.

🎥 Motion Tracking (motion_tracking.py)

Detects moving objects in video.

📦 Object Detection (object_detection.py, obj_det.py)

Detects objects from webcam or image input.

🎬 Video Object Detection (obj_detection_video.py)

Runs object detection on video files.

🛠 Installation

Clone the repository

git clone https://github.com/yourusername/mediapipe-projects.git

Move into the project folder

cd mediapipe-projects

Install required libraries

pip install opencv-python mediapipe numpy
▶️ Running the Projects

Example:

Run Air Canvas

Run Pose Detection
python pose_det.py
